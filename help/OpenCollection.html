Open Collection Or Image

Back to Index

Opens an image or collection on your computer. If the file contains a single image (Image) it will be opened and added to the current collection. If the file contains a number of images (Collection), all the images will be added to the current collection.

A list of files and directories will be displayed in the side panel. To open a file, you can either double-click a file in the list, or choose a file and click the Open Image button at the bottom. Game Imager will attempt to open the selected file using the best plugin it can find.
