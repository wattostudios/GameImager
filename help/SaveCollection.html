Save Collection

Back to Index

Saves the collection to a single file. If the chosen plugin supports multiple files, all the images in the collection will be saved to a single file. Otherwise, only the first image will be saved.

A list of files and directories will be displayed in the side panel. To save a file, enter a name for the new collection, and click the Save Collection button at the bottom. Game Imager will save the images in the format chosen in the bottom box.
