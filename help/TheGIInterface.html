The Game Imager Interface

Back to Index

The interface of Game Imager only contains 4 main areas, which keeps it simple, and removes the need for multiple screens and large popup boxes.

1. The big box in the middle is the ImageList. When you open an image, the image list will display the image for you.

ImageList


2. The list on the left is the ThumbnailList. Thumbnailis of all the images in the current collection will be displayed here. You can use this list to choose the image you want to display.

ThumbnailList


3. The most important and powerful section is the box on the right: the SidePanel. The SidePanel is used to perform all the main functions of Game Imager, including reading and writing images. No matter what you need to do, the SidePanel will always be the place to go.

SidePanel


4. The buttons and menus at the top load different screens into the SidePanel. For example, if you want to open an image, click the 'Open Image' button and the SidePanel will change. You can also perform the same thing using the menus.
