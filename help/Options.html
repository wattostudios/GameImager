Options

Back to Index

Allows you to set a variety of Game Imager properties. Choose a option group from the box at the top, and the side panel will change to show all the options in that group. When you have finished changing the options, click the Save Options button at the bottom.


Options Button

The options are categorized as follows:

Appearance: Change the color and appearance of the program
Directories: The directories used for various things such as exporting and saving.
General: Basic options that don't fit in to any other category
Opening Images: Settings for opening images and collections, and for affecting the way Game Imager determines the correct plugin for opening an image.
Popups: Enable or disable the popups that appear when performing a task
Toolbar Changer: Change and re-order the buttons that are shown on the toolbar

If you move the mouse over an option, a more details description of the option will be shown in the statusbar at the bottom of the screen.
