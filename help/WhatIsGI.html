What is Game Imager?

Back to Index

Game Imager is a program that reads and writes images used in games. In many games, the images are not a standard format, so you can't use a normal image editing program to change them. So, with the assistance of Game Imager, you are able to convert these game images into different formats, or just view them in the program.

Game Imager was build to allow both individual and mass conversion of images. You can create a new collection of images and save them as a single file, or convert them all into a different format. Conversion can be done into a variety of different formats, including many generic image types like BMP and JPEG, which allows you to edit the images in external programs like Photoshop or Gimp.

Another important feature is the use of plugins. Until now, you needed to download a whole new program when a new image format was developed. With Game Imager, all you need to do is download a small plugin which works together with the program to allow access to these new formats. The plugin system has been developed to be specifically simple, allowing almost anyone to develop a new plugin without great programming knowledge.

Game Imager also comes with some additional functions to make your life easier, such as displaying and changing the colors in an image, and zoom support. The images in the collection are shown as thumbnails so you can visually see what the images look like.
